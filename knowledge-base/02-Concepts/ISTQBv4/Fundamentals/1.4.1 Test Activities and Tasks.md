**K2**:
There are common sets of test activities without which testing is less likely to achieve test objectives. **===>** **Test Process**
A **test process** is tailored to a given situation. *Test activities* included are decided as part of the **test planning**.

ISO/IEC/IEEE 29119-2 standard

**Test Planning**
- define test objectives and select best approach to achieve them within constraints imposed by context

**Test Monitoring and Control**
- **monitoring**: involves ongoing checking of all activities and comparing progress to plan
- **control**: involves taking necessary actions to meet testing objectives

**Test Analysis**
- answers the question *what to test?* 
- includes analyzing test basis to identify testable features
- define and prioritize test conditions with risks and risk levels
- test basis and test objects are evaluated for possible defects and assess their testability

**Test Design**
- answers the question *how to test?*
- elaborate test conditions into test cases and test charters
- identify coverage items for test inputs
- define test data requirements
- design test environment
- identify required infrastructure and tools

**Test Implementation**
- includes acquiring testware for test execution
- test cases are organized into test procedures and test suites
- manual and automated test script creation
- prioritize and arrange test procedures within execution schedule for efficient test execution

**Test Execution**
-  run tests in accordance with test execution schedule(test runs)
- test execution may take many forms, can be manual, automated, continuous or pair sessions
- test results are compared with expected results and get logged
- anomalies are analyzed for their root cause and get reported 

**Test Completion**
- these usually occur at project milestones(release, end of iteration, test level completion)
- for any unresolved defects, change requests and/or product backlog items
- any testware is identified and archived for later use
- test environment is shut down on agreement state
- test activities analyzed for lessons and improvements for later
- test completion report is created and communicated
